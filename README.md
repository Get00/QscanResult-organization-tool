# Qscan 结果解析工具

一个用于解析和可视化 Qscan 扫描结果的图形界面工具，支持多种格式的扫描结果文件，并提供便捷的数据筛选、导出和分析功能。

## 功能特性

- **多格式支持**：支持 JSON、TXT、CSV 格式的 Qscan 扫描结果文件
- **三标签页界面**：总览、弱口令、未授权访问三个独立视图
- **智能搜索与过滤**：支持按关键词和特定服务类型进行数据过滤
- **右键菜单功能**：支持导出选中数据、复制 IP/端口/URL 等操作
- **批量导出**：支持将数据导出为 CSV 格式
- **统计信息**：显示文件名称、记录数量、唯一主机数等统计信息
- **自适应列宽**：支持调整表格列宽以适应内容

## 界面说明

### 总览标签页
- 显示完整的扫描结果，包含以下列：
  - URL：目标地址
  - 服务：服务类型（如 http、https、ssh 等）
  - 关键词：扫描关键词
  - IP：目标 IP 地址
  - 端口：开放端口
  - 长度：响应长度
  - 操作系统：识别的操作系统
  - 产品：识别的产品
  - 版本：产品版本
  - 摘要：扫描摘要信息

### 弱口令标签页
- 专门显示弱口令扫描结果，包含：
  - URL：目标地址
  - IP：目标 IP 地址
  - 端口：开放端口
  - 服务：服务类型
  - 用户名：破解的用户名
  - 密码：破解的密码
  - 其他信息：附加信息
  - 发现时间：发现时间

### 未授权访问标签页
- 显示未授权访问检测结果，包含：
  - URL：目标地址
  - IP：目标 IP 地址
  - 端口：开放端口
  - 服务：服务类型
  - 发现时间：发现时间
  - 详细信息：详细描述

## 使用方法

1. **启动程序**：
   ```bash
   python qscan_parser_gui.py
   ```

2. **加载扫描结果**：
   - 点击"选择扫描结果文件"按钮
   - 选择对应的 Qscan 扫描结果文件
   - 如需要，可手动选择文件类型（自动检测、JSON、TXT、CSV）

3. **数据筛选**：
   - 在搜索框中输入关键词进行全局搜索
   - 使用服务类型下拉框筛选特定服务

4. **数据导出**：
   - 导出全部数据：点击对应标签页的"导出CSV（全部）"按钮
   - 导出选中数据：右键选择行，点击"导出选中行到CSV"

5. **快捷操作**：
   - 右键单击表格行可复制 IP、端口、URL 等信息
   - 支持多选操作
   - 点击"自适应列宽"按钮调整列宽

## 右键菜单功能

- **导出选中行到CSV**：将选中的行导出为 CSV 文件
- **复制IP**：复制选中行的 IP 地址
- **复制端口**：复制选中行的端口号
- **复制URL**：复制选中行的 URL
- **复制IP+端口**：复制选中行的 IP:端口 格式
- **复制整行**：复制选中行的所有数据
- **全选（当前表格）**：选择当前表格的所有行

## 系统要求

- Python 3.x
- tkinter（通常随 Python 一起安装）
- csv 模块
- re 模块

## 支持的文件格式

- **JSON 文件**：标准 Qscan JSON 格式
- **TXT 文件**：纯文本格式的扫描结果
- **CSV 文件**：逗号分隔值格式
- **自动检测**：自动根据文件扩展名判断格式

## 文件编码支持

工具支持多种文本编码格式，包括：
- UTF-8
- UTF-8-BOM
- GBK
- Latin-1

## 统计信息

工具会显示以下统计信息：
- 导入的文件名称
- 总记录数
- 唯一 Host:Port 数量
- Web 服务数量 (http/https)
- 弱口令结果数量
- 未授权访问结果数量

## 注意事项

1. 对于大型扫描结果文件，加载可能需要一些时间
2. 建议使用过滤功能来快速定位特定类型的数据
3. CSV 导出文件使用 UTF-8-BOM 编码，确保中文正常显示
4. 右键菜单支持多选操作，可以同时处理多个条目

## 故障排除

- **文件加载失败**：检查文件格式是否正确，确认文件未被其他程序占用
- **数据显示异常**：尝试使用不同的文件编码选项
- **界面响应缓慢**：对于大量数据，考虑使用过滤功能减少显示内容
